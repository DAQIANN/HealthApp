# Healthcare Plan Explorer

A TypeScript React webapp skeleton for exploring historical healthcare data, comparing plans, and evaluating cost effectiveness.

## Features

- React + TypeScript + Vite starter
- Dark mode / light mode toggle
- No user login or tracking
- Sample dashboard for plan comparison

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal.

## Database and backend placeholders

This project includes a placeholder server folder for future PostgreSQL integration.

- Use `DATABASE_URL` in `.env` for PostgreSQL connection configuration
- Add tables for plans, services, conditions, treatments, and historical metrics
- Build API routes for service comparison, historical analysis, and search queries
